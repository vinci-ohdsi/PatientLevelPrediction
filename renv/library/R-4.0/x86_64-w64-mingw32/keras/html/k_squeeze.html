k_squeeze {keras}R Documentation

Removes a 1-dimension from the tensor at index axis.

Description

Removes a 1-dimension from the tensor at index axis.

Usage

k_squeeze(x, axis)

Arguments

x

A tensor or variable.

axis

Axis to drop (axis indexes are 1-based).

Value

A tensor with the same data as x but reduced dimensions.
