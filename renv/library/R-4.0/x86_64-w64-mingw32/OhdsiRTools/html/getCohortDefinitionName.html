getCohortDefinitionName {OhdsiRTools}R Documentation

Get a cohort definition's name from WebAPI

Description

Get a cohort definition's name from WebAPI

Usage

getCohortDefinitionName(baseUrl, definitionId, formatName = FALSE)

Arguments

baseUrl

The base URL for the WebApi instance, for example: "http://server.org:80/WebAPI".

definitionId

The cohort definition id in Atlas.

formatName

Should the name be formatted to remove prefixes and underscores?

Details

Obtains the name of a cohort.

Value

The name of the cohort.
