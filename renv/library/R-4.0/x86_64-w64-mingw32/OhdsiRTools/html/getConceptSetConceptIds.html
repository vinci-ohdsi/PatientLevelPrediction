getConceptSetConceptIds {OhdsiRTools}R Documentation

Get Concept Set Concept Ids

Description

Get Concept Set Concept Ids

Usage

getConceptSetConceptIds(baseUrl, setId, vocabSourceKey = NULL)

Arguments

baseUrl

The base URL for the WebApi instance, for example: "http://server.org:80/WebAPI".

setId

The concept set id in Atlas.

vocabSourceKey

The source key of the Vocabulary. By default, the priority Vocabulary is used.

Details

Obtains the full list of concept Ids in a concept set.

Value

A list of concept Ids.
