insertConceptSetConceptIdsInPackage {OhdsiRTools}R Documentation

Insert a set of concept sets' concept ids into package

Description

Insert a set of concept sets' concept ids into package

Usage

insertConceptSetConceptIdsInPackage(fileName, baseUrl)

Arguments

fileName

Name of a CSV file in the inst/settings folder of the package specifying the concept sets to insert. See details for the expected file format.

baseUrl

The base URL for the WebApi instance, for example: "http://server.org:80/WebAPI".

Details

The CSV file should have:

atlasId

The concept set Id in ATLAS.
